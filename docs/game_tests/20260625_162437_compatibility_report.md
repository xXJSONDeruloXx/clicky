## Technical Notes

All games are `eapp` format (ARM executables). The experimental GL HLE renderer successfully decodes OpenGL ES command streams for most titles. Games in FAIL category may have:
- Different OpenGL ES command patterns not yet handled
- Additional initialization requirements
- Different texture/pixel formats

## Environment

```bash
CLICKY_EXPERIMENTAL_GL_HLE=1
CLICKY_GL_GATE_B=1
CLICKY_GL_LIVE_CONTINUOUS=1
CLICKY_GL_PRESENT_VFLIP=1
RUST_LOG=EAPP_GL=info
```
