**Status:** ❌ NO GFX | **Draws:** 0 | **Engine:** iQuiz Engine

## Quick Start
```bash
# Renders nothing - textures never loaded
./target/release/eapp /Users/kurt/Downloads/16-ipod-games/Games_RO/11002
```

## Issue
Game uses `Filesytem` import module to initialize filesystem, then tries to
load textures via `AsyncFileIO:7` (directory enumeration). The emulator
handles `AsyncFileIO:3` (file read) but not `AsyncFileIO:7` (directory list).

As a result, TWA's 126 `.ipd` texture files and 26 `.tga` files are never
loaded, and all 8,273 draw attempts are skipped.
